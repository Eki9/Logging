# Logging and Tally Counter REST API

The purpose of this project is to demonstrate structured logging and basic REST API design.

## Technologies Used

- Node.js v18
- Express v4.18.2
- Winston v3.11.0
- Mocha (for testing)
- Chai (for assertions)
- Sinon (for spies/stubs)


## Setup

1. **Clone repository**
``` bash
git clone https://github.com/Eki9/Logging.git
cd Logging
```
2. **Install dependencies**
```bash
npm install
```
3. **Run Code**
``` bash
node src/main.js
```
4. **Run tests**
``` bash
npm test
```
